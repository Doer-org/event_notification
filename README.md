# event_notification_bot

## Doer-org/idea

[おしゃべり定期通知Bot #24 ](https://github.com/Doer-org/idea/issues/24)

discordのイベント(もくもく会や定期LT会など)の通知をするbot

## Getting Started

```
docker-compose up -d --build
```

```
docker-compose exec python3 bash
```

```
python hello.py
```

```hello docker!``` と表示されたらok
